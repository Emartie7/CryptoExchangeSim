void MerkelMain::init ( bool  debug)

Initialize and run MerkelMain exchange simulator.

Verifies that orderbook data has been properly loaded. Adds an initial amount of currency to the users wallet to allow trading.

Allows user to specify whether application should be started in "debug" mode, which will initialize application class and underlying wallet, but NOT enter the main loop (i.e. user input will not be acepted from terminal).

Parameters
debugBoolean flag to indicate whether application should run in debug mode.
