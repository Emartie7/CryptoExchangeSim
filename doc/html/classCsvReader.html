std::vector< OrderBookEntry > CsvReader::readCSV ( std::string  csvFilename)
static

Reads a CSV file containing orderbook entry data into a vector.

Uses an input file stream to read a CSV file. Each line is tokenised into orderbook entry objects. If the line does not contain the required amount of tokens as specified by ORDERBOOK_ENT_NTOKENS, the line is skipped.

See also
OrderBookEntry class for the expected data and its format.
Parameters
csvFilenamepath to CSV file to be used as input.
Returns
A vector of orderbook entry objects read in from file.
